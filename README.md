# Skills

A collection of Claude Code skills.

## Available Skills

### web-search
Perform web searches using Baidu search engine via the `baidusearch` Python library.

- **Directory**: `web-search/`
- **Dependency**: `pip install baidusearch`
- **Usage**: Automatically invoked when searching the web

## Installation

1. Clone this repository:
```bash
git clone https://github.com/jacksmion/skills.git
```

2. Copy individual skill directories to your Claude Code skills directory

## Usage

Each skill is automatically invoked by Claude Code based on context and user requests. See individual skill directories for more details.
